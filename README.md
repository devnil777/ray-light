# Ray Light — Инструмент анализа и обработки изображений

Ray Light — это веб-приложение для профессионального анализа изображений, позволяющее сравнивать различные эффекты и фильтры в реальном времени с использованием сетки.

## Основные возможности

### 1. Гибкая сетка (Grid)
- Поддержка различных конфигураций: от одного окна до сетки 4x4.
- Возможность применять разные эффекты к одному и тому же изображению в разных ячейках для детального сравнения.

### 2. Режимы масштабирования (Zoom)
- **Авто-режим:** Изображение автоматически вписывается в размер ячейки.
- **Ручной режим:** Вы можете свободно увеличивать (колесо мыши) и перемещать (перетаскивание мышью) изображение.
- **Сохранение состояния:** В ручном режиме масштаб и позиция сохраняются при переключении между фотографиями.
- **Сброс:** Двойной клик по изображению возвращает его в авто-режим.

### 3. Эффекты и анализ
- **Зоны фокуса (Focus Peaking):** Подсветка синим цветом самых резких и контрастных участков изображения.
- **Гистограмма:** Четырехканальный вывод (Яркость, Красный, Зеленый, Синий) для оценки экспозиции.
- **Круг Иттена:** Анализ цветового распределения по 12 секторам. Программа рассчитывает процентное содержание каждого цвета, исключая нейтральные (серые) оттенки.
- **Технические фильтры:** Каналы (R, G, B), инверсия, экспозиция, поиск пересветов и теней (Clipping).

### 4. Интерфейс
- Тёмная тема в стиле профессиональных редакторов (Lightroom / VS Code).
- Управление горячими клавишами (стрелки влево/вправо для навигации).
- Сортировка эффектов методом Drag-and-Drop.

## Как запустить

1. Установите зависимости:
   ```bash
   pip install fastapi uvicorn Pillow
   ```

2. Запустите сервер, указав папку с вашими изображениями:
   ```bash
   python3 server.py ./путь/к/папке/с/фото
   ```

3. Откройте браузер по адресу: `http://localhost:8000`

## Использование
- Выберите формат сетки в правой панели.
- Нажимайте на кнопки в разделе «Доступные эффекты», чтобы добавить их в активные ячейки.
- Перетаскивайте эффекты в списке, чтобы изменить их порядок в сетке.
- Используйте колесо мыши для зума и левую кнопку для панорамирования.
- Клавиши **←** и **→** для быстрого просмотра всей папки.
