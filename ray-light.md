Приложение для проверки фотографий после ретуши перед публикаций для поиска артефактов

Суть приложения, наложение специфически эффектов на изображения, которые помогли бы визуализировать разные дефекты обработки

SPA приложение, вся логика реализуется на vanilla js/html/css
Backend выступает простейшее python приложение которое отдает html/css/js и файлы изображения на фронт (в последствие возможно будет заменено на electron)

Приложение профессиональное, без эффектов, стиль visual code/ligthroom

Структура: 
основная рабочая область где отображается фотография с различными эффектами

справа панель управления
панель управления состоит из основных настроек:
- цвет фона рабочего пространства (от белого до черного 6 градаций)
- поворот фото (90, 180, 270). Flip вертикальный, горизонтальный
- сетка (1, 2x2, 3x2, 8x8)
- список эффектов (по порядку для каждой копии изображения):
	- у эффекта опционально могут быть параметры (в формате key=value)
	- можно менять порядок эффектов
	- можно удалить эффектов
	- список ограничен кол-во изображений в сетке
	- можно использовать несколько раз один эффект с разными параметрами
* под палитра эффектов: список доступных эффектов, из него можно выбрать эффекты и добавить их в список эффектов
* ширина панели должна изменяться с помощью мыши

Т.е. на каждое изображение накладывается один свой эффект, и так для каждого изображения из сетки. Если эффектов меньше, то изображения не отображается

возможные эффекты:
- изображение как есть
- изображение разбито на 3 канала (RGB) - эффект показывает один выбранный канал
- инверсия изображения
- сдвиг экспозиции (в + или -). Параметр: ступень
- отображение пересветов и черных провалов (где потеря информации)

Кнопки перейти к следующему изображению, к предыдущему

Технические моменты:
* изображения кэшируются (с наложенными эффектами) 2 вперед, 2 назад
* наложение эффектов происходит асинхронно и отображается по мере готовности
* все эффекты должны быть представлены как модули и вынесены в отдельный файл (в будущем будут еще эффекты)
* эффект на вход получает параметры
* эффект может наложить статусную строку на изображение, например, скорость формирования изображения
* текущий набор эффектов сохроняется в localstorage
